FISCAL – THIRD PARTY TAX PROFILE
================================

Model: ThirdPartyTaxProfile
---------------------------

Fields:
- company: ForeignKey(Company)
- afip_category: CharField(10, choices=AFIP_CATEGORY_CHOICES)

VAT:
- vat_21: BooleanField(default=True)
- vat_105: BooleanField(default=False)
- vat_27: BooleanField(default=False)
- vat_exempt: BooleanField(default=False)
- vat_non_taxed: BooleanField(default=False)

IIBB:
- iibb_rate: DecimalField(max_digits=5, decimal_places=2, null=True, blank=True)
- iibb_exempt: BooleanField(default=False)

Ganancias:
- ganancias_agent: BooleanField(default=False)

Percepciones / Retenciones:
- perceptions_enabled: BooleanField(default=False)
- retentions_enabled: BooleanField(default=False)

String representation:
- __str__ = "{company.legal_name} – {afip_category}"
